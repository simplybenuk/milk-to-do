
# Supabase Functions Directory

This directory contains serverless edge functions deployed to Supabase.

Each subdirectory typically represents a distinct function endpoint.

Key functions:
- `create-checkout/`: Function for creating Stripe checkout sessions
- `decay-skip-counts/`: Function for decaying task skip counts
- `send-contact-email/`: Function for sending contact form emails
- `stripe-webhook/`: Function for handling Stripe webhook events
