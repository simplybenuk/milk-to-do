
# Task Actions Directory

This directory contains specific task-related actions for the Zustand task store.

Each file typically represents a specific action or group of related actions.

Key files:
- `addTask.ts`: Actions for adding new tasks
- `completeTask.ts`: Actions for completing tasks
- `deleteTask.ts`: Actions for deleting tasks
- `fetchTasks.ts`: Actions for fetching tasks from the database
- `focusModeActions.ts`: Actions related to focus mode functionality
- `updateTask.ts`: Actions for updating existing tasks
- `updateTaskPriority.ts`: Actions for updating task priority
- `skipTask.ts`: Actions for skipping tasks
- `refreshTaskExpiryDate.ts`: Actions for refreshing task expiry date
- `decayActions.ts`: Actions related to task decay functionality
