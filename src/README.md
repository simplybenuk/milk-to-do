
# Source Directory

This is the main source directory for the SourList application. It contains the application's React components, hooks, stores, and utility functions.

Key subdirectories:
- `components/`: UI components used throughout the app
- `context/`: React context providers
- `hooks/`: Custom React hooks
- `integrations/`: External service integrations (e.g., Supabase)
- `lib/`: Utility functions and libraries
- `pages/`: Page-level components for routing
- `stores/`: State management using Zustand
- `types/`: TypeScript type definitions
