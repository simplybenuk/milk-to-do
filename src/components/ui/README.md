
# UI Components

This directory contains base UI components built with Shadcn UI and Radix UI primitives. These components provide the foundation for the SourList application's user interface.

These components include:
- Button
- Card
- Dialog
- Form elements
- Dropdown and select components
- Toast notifications
- and more

Most components are built using the Shadcn UI library, which provides a clean, accessible, and customizable component library based on Radix UI primitives.
