
# Chart UI Components

This directory contains components for creating and styling charts used in the statistics section of the application.

Key files:
- `ChartContainer.tsx`: Container component for chart elements
- `ChartContext.tsx`: React context for sharing chart configuration
- `ChartTooltip.tsx`: Tooltip component for chart data points
- `ChartLegend.tsx`: Legend component for charts
- `ChartStyle.tsx`: Styling utilities for charts
- `types.ts`: TypeScript definitions for chart components
