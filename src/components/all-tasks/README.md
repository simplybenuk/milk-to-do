
# All Tasks Components

This directory contains components for the "All Tasks" view in the SourList application. This view displays all tasks in a list format, allowing users to see all their tasks at once.

Key files:
- `AllTasksList.tsx`: Main component for the all tasks view
- `TaskList.tsx`: Reusable component for rendering a list of tasks
- `useAllTasksView.ts`: Hook for managing the all tasks view state and logic
