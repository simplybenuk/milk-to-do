
# Focus Mode Components

This directory contains components related to the Focus Mode feature, which helps users concentrate on one task at a time.

Key files:
- `FocusModeContainer.tsx`: Container component for focus mode
- `FocusModePage.tsx`: Page component for focus mode
- `FocusTagSelector.tsx`: Component for selecting tags in focus mode
- `FocusTagInfo.tsx`: Component for displaying tag information in focus mode
