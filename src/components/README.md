
# Components Directory

This directory contains all the reusable UI components used throughout the SourList application.

Key subdirectories:
- `all-tasks/`: Components for displaying all tasks view
- `auth/`: Authentication-related components
- `contact/`: Contact form and related components
- `current-task/`: Components for the current task view in focus mode
- `focus/`: Components for the focus mode feature
- `layout/`: Layout components like NavBar, Footer, and containers
- `settings/`: Settings-related components
- `stats/`: Components for displaying task statistics
- `task-item/`: Components for rendering individual task items
- `ui/`: Shadcn UI components and base UI elements
