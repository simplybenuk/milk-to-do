
# Task Item Button Components

This directory contains button components used within task items for various actions.

Key files:
- `CompleteButton.tsx`: Button for completing a task
- `DeleteButton.tsx`: Button for deleting a task
- `EditButton.tsx`: Button for editing a task
- `SplitButton.tsx`: Button for splitting a task into subtasks
- `TaskMenu.tsx`: Dropdown menu component for task actions
