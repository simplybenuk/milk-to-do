
# Task Item Components

This directory contains components related to rendering individual task items in the SourList application.

Key files:
- `TaskItem.tsx`: The main task item component
- `TaskContent.tsx`: Component for displaying task content
- `TaskAgeIndicator.tsx`: Component to show visual indicators of task age
- `TaskAgeUtils.tsx`: Utility functions for determining task age styling
- `TaskMetadata.tsx`: Component for displaying task metadata
- `TextWithLinks.tsx`: Component for rendering text with clickable links
- `PriorityBadge.tsx`: Component for displaying task priority
- `buttons/`: Subdirectory containing task action buttons
