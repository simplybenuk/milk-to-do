
# Pages Directory

This directory contains top-level page components used for routing in the SourList application.

Each file typically represents a distinct route in the application. These components are responsible for layout and composition of various child components.

Key pages:
- `Index.tsx`: Main application page
- `Landing.tsx`: Landing page for non-authenticated users
- `Auth.tsx`: Authentication page
- `Settings.tsx`: User settings page
- `About.tsx`, `Contact.tsx`, `FAQ.tsx`, `Privacy.tsx`: Information pages
