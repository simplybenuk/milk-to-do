
# Hooks Directory

This directory contains custom React hooks used throughout the SourList application.

These hooks encapsulate and reuse stateful logic, making components cleaner and more focused on rendering.

Key hooks:
- `useAppView.ts`: Manages the current app view state
- `useFocusTasks.ts`: Manages tasks in focus mode
- `useTaskNavigation.ts`: Handles task navigation in focus mode
- `useTaskActions.ts`: Provides actions for manipulating tasks
- `useTaskSkipActions.ts`: Handles task skip functionality
- `useFocusModeHandlers.ts`: Manages focus mode state transitions
- `useSubscription.ts`: Manages subscription state
